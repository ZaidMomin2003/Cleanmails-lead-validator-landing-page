CleanMails is a self-hosted email verification tool. You install it on your own VPS, upload email lists or use the API, and it performs 8-step deep verification — syntax, MX, DNS, SMTP handshake, catch-all, disposable, role-based, and free-mail detection. Your data never leaves your server.

One command. Run the install script with your license key and you're live in under 3 minutes. It auto-installs Docker, PostgreSQL, Redis, sets up SSL, and creates your admin account — no manual configuration needed.
